# FPlug
FPlug是一个Fiddler插件，提供Web开发中可能使用到的一系列工具
# 版本
v1.0.0
# 环境要求
Fiddler版本要求4.6以上，推荐5.0版本以上<font color="#ff0000">(开发过程中是以Fiddler 5.0版本为基准)</font>，同时要求.NET Framework 4.6.1以上
# 下载安装
下载dist中的两个dll文件，然后复制到Fiddler安装目录中的Scripts文件夹。
# 特性
* HOST映射
* 文件映射
* HTTPS 转 HTTP
* 显示ServerIP
* 禁止缓存
* console日志
* UA模拟
* JS注入
